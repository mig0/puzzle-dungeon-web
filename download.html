Instructions to download, install and run

Source code

git checkout https://github.com/mig0/dungeon.git

Dependencies

You need at least pygame and pgzero.

On Fedora, python3-pygame-2.6.1 and pgzero-1.2.1 rpms are found to work fine. Try:

sudo dnf install python3-pygame pgzero

How to install

No installation is currently needed to quickly run the game, just checkout from github and run.

How to run

./dungeon
